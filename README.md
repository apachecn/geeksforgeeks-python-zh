## 下载

### Docker

```
docker pull apachecn0/geeksforgeeks-python-zh
docker run -tid -p <port>:80 apachecn0/geeksforgeeks-python-zh
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install geeksforgeeks-python-zh
geeksforgeeks-python-zh <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g geeksforgeeks-python-zh
geeksforgeeks-python-zh <port>
# 访问 http://localhost:{port} 查看文档
```
